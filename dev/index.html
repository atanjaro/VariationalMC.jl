VariationalMC.jl

Documentation for VariationalMC.jl. This package impelments the variational Monte Carlo (VMC) method for Hubbard and electron-phonon interactions (coming soon).

This code is currently the experimental stage of development. Use with caution.

Funding

Installation

To install the VariationalMC.jl, simply open the Julia REPL and run the commands

julia> ]
pkg> add VariationalMC

or equivalently via Pkg do

julia> using Pkg; Pkg.add("VariationalMC")

Notable External Package Dependencies

This section reviews some notable package dependencies.

  • LatticeUtilties.jl: Package that is used to represent arbitrary lattice geometries.
  • OrderedCollections.jl: Package that implements associative containers that preserve the order of insertion.
  • JLD2.jl: Package used to write data to binary files in an HDF5 compatible format.
  • CSV.jl: Package used for handling delimited text data.
